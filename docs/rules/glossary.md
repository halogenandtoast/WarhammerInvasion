# Glossary

Alphabetical lookup. Cross-references point to the file with the full
explanation.

| Term | Meaning |
|---|---|
| **Action** | Optional, bold-`Action:` effect. Triggerable in any action window. See [effects.md](./effects.md). |
| **Action window** | A pause between phase steps where either player may take actions. Engine advances only after both pass consecutively. See [turn-sequence.md](./turn-sequence.md). |
| **Active player** | The player whose turn it currently is. |
| **Applied damage** | Damage committed to a target. Toughness fires here. See [combat.md](./combat.md). |
| **Assigned damage** | Damage placed near a target but not yet committed. Can still be cancelled. |
| **Attachment** | A support card attached to another card. Sacrificed when the host leaves play. |
| **Battlefield** | The zone units must be in to attack. Base power 0. |
| **Burn token** | Placed on a capital section when it is overrun. Two burns on the same player ends the game. |
| **Capital** | A player's three-zone area: kingdom + quest zone + battlefield. |
| **Cannot** | Absolute prohibition. Overrides anything that says otherwise. See [effects.md](./effects.md). |
| **Constant effect** | Card text without a bold trigger; in effect while the card is in play. |
| **Controller** | The player currently controlling a card. Defaults to owner; can change via effects. |
| **Corrupt** | A card flagged so it cannot attack or defend. Restored at most one per kingdom phase. See [effects.md](./effects.md). |
| **Corresponding zone** | Another player's zone with the same name as the referenced card's zone. |
| **Counterstrike N** | Defender-only keyword: deal N uncancellable damage immediately on being declared defender. See [combat.md](./combat.md). |
| **Damage token** | Represents inflicted damage on units or capital sections. |
| **Decking out** | Having zero cards in deck — immediate elimination. |
| **Defender** | A unit declared in combat step 3 to defend the attacked zone. |
| **Development** | Face-down card placed in a zone during Capital phase. Each adds +1 HP to its zone. Max one per turn. See [cards.md](./cards.md). |
| **Destruction** | Faction grouping: Chaos, Orc, Dark Elf. |
| **Discard pile** | Where cards go when destroyed, sacrificed, or after a tactic resolves. |
| **Draft format card** | Special card used only during draft mode; never goes in a deck. |
| **First player penalty** | The starting player skips Quest and Battlefield phases on turn 1. |
| **Forced effect** | Bold-`Forced:` effect; fires automatically, cannot be cancelled. |
| **Hit points (HP)** | Damage capacity. Units: printed value. Zones: 8 + developments. |
| **In response** | Triggering an action during another action's resolution; resolves first (LIFO). |
| **Inflicted damage** | The total amount before assigning to targets. |
| **Kingdom** | Zone that produces resources. Base power 3. |
| **Keyword** | Shorthand rule like Toughness, Counterstrike, Scout, Limited, Order Only, Destruction Only. |
| **Limited** | Keyword: max one card with Limited played per turn (across all your Limited cards). |
| **Loyalty** | Variable cost portion paid in resources; reduced by matching race symbols in play, floor of 0. See [cards.md](./cards.md). |
| **Mulligan** | Optional one-time redraw of the starting hand during setup. |
| **Neutral** | Card with no race symbol (grey border). Usable by either faction unless text restricts it. |
| **Order** | Faction grouping: Empire, Dwarf, High Elf. |
| **Order Only / Destruction Only** | Keyword restricting a (usually neutral) card to one faction. |
| **Owner** | The player whose deck the card was in at game start. Cards revert to their owner's stack when leaving play. |
| **Power icon** | Resource symbol on a card. In Kingdom → resources, Quest → draw, Battlefield → damage. |
| **Quest** | Card type representing a long-term pursuit. Lives in the quest zone, takes one questing unit, accumulates resources for its effect. |
| **Quest zone** | Zone that draws cards. Base power 1. |
| **Race symbol** | Faction badge on a card; capital boards count too. Reduces loyalty cost of matching cards. |
| **Resource token** | Used to pay card costs and accumulate on quests. |
| **Restore** | Un-corrupt a card. One restoration per kingdom phase. |
| **Sacrifice** | Send a card to its owner's discard pile. Absolute; cannot be cancelled. |
| **Scout** | Keyword: surviving Scout units force opponent to discard random cards after damage applies. |
| **Support** | Persistent card type for buildings, locations, weapons, attachments. |
| **Tactic** | One-shot card. Played from hand, effect resolves, goes to discard. |
| **Target** | The card or zone an effect names. Removing the target before resolution may prevent the effect. |
| **Toughness N** | Keyword: cancel N damage during the apply step. Stacks additively. |
| **Trait** | Flavor tag (Warrior, Hero, Spell, Building, …). No inherent rules. |
| **Uncancellable damage** | Damage that bypasses Toughness and damage-cancellation effects (e.g. Counterstrike). |
| **Unique card** | Card with a banner before its title. A player may not control more than one copy at a time. Both players can each control one. |
| **Unit** | Persistent card type for heroes, soldiers, monsters. Attack, defend, quest, produce. |
| **Zone** | One of Kingdom, Quest, Battlefield. Each player has three. Each starts at 8 HP. |
