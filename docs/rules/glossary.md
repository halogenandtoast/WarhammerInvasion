# Glossary

Alphabetical lookup. Cross-references point to the file with the full
explanation.

| Term | Meaning |
|---|---|
| **Action** | Optional, bold-`Action:` effect. Triggerable in any action window. See [effects.md](./effects.md). |
| **Action window** | A pause between phase steps where either player may take actions. Engine advances only after both pass consecutively. See [turn-sequence.md](./turn-sequence.md). |
| **Active player** | The player whose turn it currently is. |
| **Ambush** | Keyword on developments. Only triggerable on a face-down development. |
| **Applied damage** | Damage committed to a target. Toughness fires here. See [combat.md](./combat.md). |
| **Assigned damage** | Damage placed near a target but not yet committed. Can still be cancelled or redirected. |
| **Attachment** | A support card attached to another card. Sacrificed when the host leaves play. See [cards.md](./cards.md). |
| **Battlefield** | The zone units must be in to attack. Base power 0. |
| **Beginning of the Turn** | Phase 0 (FAQ 2.2). All "at the beginning of your turn" effects fire here, before Kingdom. |
| **Burn token** | Placed on a capital section when it is overrun. Two burns on the same player ends the game. |
| **Cancel** | To prevent assigned damage from being applied. Cancelled tokens return to the pool. |
| **Capital** | A player's three-zone area: kingdom + quest zone + battlefield. |
| **Cannot** | Absolute prohibition. Overrides anything that says otherwise. See [effects.md](./effects.md). |
| **Conditional action** | An Action that creates a constant effect waiting for a trigger condition; expires at end of turn or when condition met. See [timing.md](./timing.md). |
| **Constant effect** | Card text without a bold trigger; in effect while the card is in play. |
| **Controller** | The player currently controlling a card. Defaults to owner; can change via effects. |
| **Corrupt** | A card flagged so it cannot attack or defend. Restored at most one per kingdom phase. See [effects.md](./effects.md). |
| **Corresponding zone** | Another player's zone with the same name as the referenced card's zone. |
| **Counterstrike N** | Defender-only keyword: deal N uncancellable damage immediately on being declared defender. See [combat.md](./combat.md). |
| **Cost** | The number in the top-left of the card. Loyalty is *not* part of cost for effect references. |
| **Damage token** | Represents inflicted damage on units or capital sections. |
| **Decking out** | Having zero cards in deck — immediate elimination. |
| **Defender** | A unit declared in combat step 3 to defend the attacked zone. |
| **Destroyed** | A card is removed from play and placed in its owner's discard pile. Distinct from sacrifice. |
| **Destruction** | Faction grouping: Chaos, Orc, Dark Elf. |
| **Development** | Face-down card placed in a zone during Capital phase. Each adds +1 HP to its zone. Max one per turn. See [cards.md](./cards.md). |
| **Discard pile** | Where cards go when destroyed, sacrificed, or after a tactic resolves. |
| **Draft format card** | Special card used only during draft mode; never goes in a deck. |
| **End of the Turn** | Phase 5 (FAQ 2.2). Action window, then "at the end of the turn" triggers, then UntilEndOfTurn modifiers expire. |
| **Entering play** | Moving from an out-of-play zone (deck / hand / discard) into an in-play zone. Moving between in-play zones is *not* entering play. |
| **Faceup development** | A development flipped face-up (Judgement of Loec, Mage of Loec). Loses text/power/cost/loyalty/traits/HP but keeps title and uniqueness. |
| **First player penalty** | The starting player skips Quest and Battlefield phases on turn 1. |
| **Forced effect** | Bold-`Forced:` effect; fires automatically, cannot be cancelled. |
| **Healing** | Removing damage tokens from a unit. Requires the unit to have damage on it. |
| **Hit points (HP)** | Damage capacity. Units: printed value. Zones: 8 + developments. See also *Remaining hit points*. |
| **In play** | Kingdom, quest zone, battlefield. Capital board too. |
| **In response** | Triggering an action during another action's resolution; resolves first (LIFO). |
| **Indirect damage** | Non-combat damage dealt to a *player*, who then assigns it across their own units/legends/capital. Cannot be assigned to a burning zone or in excess of what would destroy/burn. |
| **Inflicted damage** | The total amount before assigning to targets. |
| **Just played** | A card mid-resolution on an active action chain. Cards that target "just played" cards may name any card on the chain. |
| **Kingdom** | Zone that produces resources. Base power 3. |
| **Keyword** | Shorthand rule like Toughness, Counterstrike, Scout, Limited, Order Only, Destruction Only, Ambush. |
| **Leaving play** | Moving from an in-play zone to an out-of-play zone (hand / deck / discard). Triggers self-referential abilities. |
| **Legend** | Persistent card type that lives on the capital board (not in a zone). Contributes power to all three zones. Max one in play per player. Not affected by card effects that target unit/support/tactic. See [cards.md](./cards.md). |
| **Limit one Hero per zone** | Restriction on Hero units — both players blocked from putting a second Hero into the same zone of that player's capital. |
| **Limited** | Keyword: max one card with Limited played per turn (across all your Limited cards). |
| **Loyalty** | Variable cost portion paid in resources; reduced by matching race symbols in play, floor of 0. *Not* part of "cost" for effect references. See [cards.md](./cards.md). |
| **Move** | To shift a card from one zone to another. The move must end in a different zone. |
| **Mulligan** | Optional one-time redraw of the starting hand during setup. |
| **Neutral** | Card with no race symbol (grey border). Usable by either faction unless text restricts it. |
| **Non-combat damage** | Damage from card effects, not combat. Always applied as soon as it is assigned. |
| **Or** | Choice between two outcomes; the player must pick an option they can fully resolve. |
| **Order** | Faction grouping: Empire, Dwarf, High Elf. |
| **Order Only / Destruction Only** | Keyword restricting a (usually neutral) card to one faction. |
| **Out of play** | Deck, hand, and discard pile. |
| **Owner** | The player whose deck the card was in at game start. Cards revert to their owner's stack when leaving play. |
| **Power icon** | Resource symbol on a card. In Kingdom → resources, Quest → draw, Battlefield → damage. |
| **Quest** | Card type representing a long-term pursuit. Lives in the quest zone, takes one questing unit, accumulates resources for its effect. Effects can trigger on a quest even with no unit questing. |
| **Quest zone** | Zone that draws cards. Base power 1. |
| **Race symbol** | Faction badge on a card; capital boards count too. Reduces loyalty cost of matching cards. |
| **Redirect** | Re-target damage assigned to one card onto another. Happens after assign, before apply. Not cumulative. |
| **Remaining hit points** | A unit's HP minus damage currently on it; a zone's `8 + developments` minus damage on the section. |
| **Resource token** | Used to pay card costs and accumulate on quests. |
| **Restore** | Un-corrupt a card. One restoration per kingdom phase. |
| **Sacrifice** | Send a card to its owner's discard pile. Absolute; cannot be cancelled. |
| **Scout** | Keyword: surviving Scout units force opponent to discard random cards after combat — even if no damage was dealt. |
| **Search** | Look at the top N cards of a deck. Distinct from "look at." |
| **Self-referential** | Card text using "this unit"/"this card." Allows the ability to trigger even after the card has left play. |
| **Simultaneous effects** | Two non-Action effects of the same type triggering at once. Active player resolves all of his in chosen order, then opponent. |
| **Skip** | Bypass a phase. A phase can be skipped multiple times. |
| **Support** | Persistent card type for buildings, locations, weapons, attachments. |
| **Tactic** | One-shot card. Played from hand, effect resolves, goes to discard. Tactics are themselves Actions. |
| **Target** | The card or zone an effect names. Targets are validated at play and at resolution; removing one before resolution may cancel the effect. |
| **Then** | The preceding effect must resolve successfully before the "then" clause resolves. |
| **Toughness N** | Keyword: cancel N damage during the apply step. Stacks additively. |
| **Trait** | Flavor tag (Warrior, Hero, Spell, Building, …). No inherent rules. |
| **Trigger condition** | Anything that must happen before a card ability can fire. See [timing.md](./timing.md). |
| **Triggered action** | An Action with a trigger condition. Each copy fires at most once per trigger this turn. |
| **Uncancellable damage** | Damage that bypasses Toughness and damage-cancellation effects (e.g. Counterstrike, non-combat damage from certain sources). |
| **Unique card** | Card with a banner before its title. A player may not control more than one copy at a time. Both players can each control one. |
| **Unit** | Persistent card type for heroes, soldiers, monsters. Attack, defend, quest, produce. |
| **X** | When unspecified, X = 0. Variable values inside effects only. |
| **Zone** | One of Kingdom, Quest, Battlefield. Each player has three. Each starts at 8 HP. |
